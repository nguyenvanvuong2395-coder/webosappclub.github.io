Alcyone 3.0.0

Alcyone is a VPN client for rooted LG webOS TVs. It runs VPN directly on the TV through xray and tun2socks, without requiring a separate router, phone, or PC as a gateway.

The name Alcyone comes from the brightest star in the Pleiades cluster. The app follows that idea with a dark cosmic interface, violet accents, and a star-inspired logo.

Alcyone supports VLESS/XHTTP, VMess, Trojan, Shadowsocks, SOCKS5, and Hysteria2 profiles. Servers and subscriptions can be added from a phone or PC through the built-in web interface, then selected from the TV UI with the remote.

A key feature is advanced subscription import. Alcyone can try compatible client profiles, headers, and User-Agent values, which helps load subscriptions that are normally available only to specific VPN apps and are not returned to a regular browser.

Features include server selection, subscription updates, autostart, external IP check, local SVG country flags, and tunnel logs for diagnostics.

Root access is required. If you report a bug, please include your TV model and webOS version.
